# Forum IT avec Laravel

## Contexte du projet

```
L'objectif de ce projet va être de mettre en place la conception et la réalisation d'une application de forum des questions IT.
```

### fonctionnalités

```
* L'application est protégé par un système d'authentification avec 2 roles (client, admin)
* L'utilisateur client peut ajouter des questions et répondre aux questions des autres utilisateurs, et voir les questions et  les réponses qu'il a créé dans son espace perso.
* L'admin peut modérer toutes les questions et réponses sur le forum(supprimer)
* L'admin peut modérer les utilisateurs
```
